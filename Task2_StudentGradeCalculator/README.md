# Task 2 - Student Grade Calculator (Java)

## Description
This Java program calculates total marks, average percentage, and grade
based on marks entered by the user for multiple subjects.

## Features
- Accepts marks for any number of subjects
- Calculates total and average
- Assigns grade using conditions

## Concepts Used
- Scanner
- Loops
- Conditional statements

## How to Run
1. Compile: javac StudentGradeCalculator.java
2. Run: java StudentGradeCalculator
