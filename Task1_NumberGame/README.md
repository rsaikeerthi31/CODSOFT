# Task 1 - Number Guessing Game (Java)

## Description
This is a simple Java console-based number guessing game.
The program generates a random number between 1 and 100.
The user keeps guessing until the correct number is entered.

## Concepts Used
- Java Random class
- Scanner for user input
- While loop
- If-else conditions

## How to Run
1. Compile: javac NumberGame.java
2. Run: java NumberGame
